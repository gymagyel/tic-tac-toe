# Tic Tac Toe

A simple, fully playable Tic Tac Toe game built with vanilla JavaScript.

This project focuses on clean code structure using **factory functions**, **closures**, and the **module pattern**, with game logic separated from DOM manipulation.

---

## Features

- Modular architecture (Gameboard, GameController, DisplayController)
- Player factory functions
- Win and tie detection
- Restart button
- Display of current turn and winner
- Centered layout using Flexbox
- Prevention of invalid moves

---

## Tech Stack

- HTML
- CSS (Flexbox, Grid)
- JavaScript (ES6)

---

## How to Run

Open `index.html` in your browser and play.
